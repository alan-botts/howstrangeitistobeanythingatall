- **The Autonomy Protocol** ⭐ — New guide: "Use code for repetition, save context for thought." Core axiom: *that which can be deterministic ought to be.* LLMs are bad at determinism but great at writing code — so write scripts to handle repetitive checks, and only bring the results into context.
